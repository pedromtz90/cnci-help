Para practicar el uso y adaptación a la plataforma, existe el Curso de Estilos de Aprendizaje, en el cual se responde una prueba para conocer el estilo de aprendizaje con el que cuentas y que puedas optimizar tu aprendizaje. La segunda actividad dentro de este curso será la carga de una portada en WORD, con la intención de que aprendas el envío de las actividades. 

Este curso no se requiere acreditar pues solo sirve para conocer la plataforma de Blackboard Ultra, si eres alumno de primer ingreso, es clave sepas que el curso que sí deberás acreditar se llamada Curso de inducción.
